This Project is the bsp package for MiniPupper ROS.

#####################################################################################################
When get the pure ubuntu system (recommend ubuntu 20). You'd better install this packeg as following steps:
1. cd home/ubuntu
2. git cone https://github.com/mangdangroboticsclub/minipupper_ros_bsp.git
3. cd minipupper_ros_bsp/mangdang ;
4. sudo sh Legacy/pre_install.sh ;
5. sudo sh install.sh
6. reboot ;
